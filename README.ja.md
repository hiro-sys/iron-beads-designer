# Bead Pattern Maker（アイロンビーズ図案メーカー）

画像をアップロードして、またはGoogle GeminiによるAIお題入力から、パーラービーズ／ナノビーズの図案を自動生成するWebアプリケーションです。処理はすべてブラウザ内で完結します（AIお題モードを使う場合のみ、入力したお題テキストがGemini APIへ送信されます）。

## 主な機能

- 「📷 画像から」「✨ AIお題から」の2つの入力方法をいつでも切り替え可能
- 画像アップロード（JPEG／PNG／GIF／WebP、最大10MB、ドラッグ&ドロップ対応）
- パーラービーズ（100色）／ナノビーズ（55色）への自動変換（CIE76色差による最近色マッチング）
- Google Gemini によるAIお題生成（BYOK方式）: 自分のGoogle AI StudioのAPIキーを入力し、Geminiモデル（無料枠の有無でグループ表示）を選んで、お題（例:「ねこ」「ハート」「星」）を入力するだけで、画像アップロード不要でドット絵図案をAIが生成する。この方式ではお題テキストのみがGemini APIへ送信される
- APIキーはセッション中のブラウザメモリにのみ保持され、永続ストレージ（localStorage／sessionStorage／Cookie等）へは一切保存されない。ページのリロードで自動的に消去される
- プレート構成の指定（1×1〜10×10）とおすすめサイズの提案
- 背景除外、使用色の制限・減色、リサイズ方式／フィットモードの選択
- 図案の手動編集（ドラッグによる連続編集に対応）
- 使用色一覧の表示とPNG画像エクスポート

## 技術スタック

- Vanilla JavaScript（ESモジュール）
- Vite（ビルドツール）
- HTML5 Canvas API
- Google Gemini API（AIお題生成機能・BYOK方式、専用バックエンドは持たない）
- Vitest + fast-check（テスト）

## ディレクトリ構成

このリポジトリはモノレポ構成で、アプリ本体は `bead-pattern-maker/` ディレクトリにあります。

## セットアップ

```bash
cd bead-pattern-maker
npm install
```

## 開発コマンド

```bash
cd bead-pattern-maker
npm run dev      # 開発サーバーを起動
npm run build    # 本番ビルド（dist/ に出力）
npm run preview  # ビルド結果をプレビュー
npm test         # テストを実行
```

## デプロイ

AWS Amplify Hosting でホスティングします。リポジトリルートの `amplify.yml` が、モノレポ構成（`appRoot: bead-pattern-maker`）でのビルド設定を定義しています。

### セキュリティヘッダー

リポジトリルートの `customHttp.yml` が、CSP（Content-Security-Policy）・HSTS・X-Frame-Optionsなどのセキュリティヘッダーを定義しています。AWS Amplify Hostingはこのファイルを検出すると、コンソールでの事前設定なしに、初回デプロイから自動的にこれらのヘッダーを適用します（Consoleの「Custom headers」設定より本ファイルが優先されます）。Gemini APIのエンドポイントを変更したり外部リソースを追加したりする場合は、`Content-Security-Policy`ヘッダーの`connect-src`ディレクティブも合わせて更新してください。
