# Bead Pattern Maker（アイロンビーズ図案メーカー）

画像をアップロードすると、パーラービーズ／ナノビーズの図案を自動生成するWebアプリケーションです。画像変換・図案生成・エクスポートのすべての処理がブラウザ内で完結します。

## 主な機能

- 画像アップロード（JPEG／PNG／GIF／WebP、最大10MB、ドラッグ&ドロップ対応）
- パーラービーズ（100色）／ナノビーズ（55色）への自動変換（CIE76色差による最近色マッチング）
- プレート構成の指定（1×1〜10×10）とおすすめサイズの提案
- 背景除外、使用色の制限・減色、リサイズ方式／フィットモードの選択
- 図案の手動編集（ドラッグによる連続編集に対応）
- 使用色一覧の表示とPNG画像エクスポート

## 技術スタック

- Vanilla JavaScript（ESモジュール）
- Vite（ビルドツール）
- HTML5 Canvas API
- Vitest + fast-check（テスト）

## ディレクトリ構成

このリポジトリはモノレポ構成で、アプリ本体は `bead-pattern-maker/` ディレクトリにあります。

## セットアップ

```bash
cd bead-pattern-maker
npm install
```

## 開発コマンド

```bash
cd bead-pattern-maker
npm run dev      # 開発サーバーを起動
npm run build    # 本番ビルド（dist/ に出力）
npm run preview  # ビルド結果をプレビュー
npm test         # テストを実行
```

## デプロイ

AWS Amplify Hosting でホスティングします。リポジトリルートの `amplify.yml` が、モノレポ構成（`appRoot: bead-pattern-maker`）でのビルド設定を定義しています。
